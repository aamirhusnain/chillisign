
# ✍️ ChilliSign Outlook Signatures Add-in

ChilliSign Outlook Add-in enables users to create, manage, and insert professional email signatures directly within Outlook. It helps teams maintain consistent branding, improve communication, and automate signature management across the organization.

## ✨ Features

## ⚙️ How It Works

1. Open Outlook and start composing an email  
2. Launch the ChilliSign Add-in  
3. Select or create a signature template  
4. Customize details (name, role, company, links, etc.)  
5. Insert the signature into the email  
6. Send emails with consistent branding  

---

### 5. Sideload Add-in into Outlook

* Open Outlook
* Go to Add-ins → Upload My Add-in
* Upload the manifest file
